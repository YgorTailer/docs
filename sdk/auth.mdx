---
title: "Auth"
description: "Use o cliente publico do SDK para login, cadastro, OAuth e sessoes."
---

## Criar cliente publico

```typescript
import { createClient } from '@nivo-lat/sdk'

const client = createClient('https://api.nivo.lat', 'nivo_pub_...')
```

Use Publishable Key no frontend. Secret Key fica somente no backend.

## Cadastro

```typescript
const { data, error } = await client.auth.signUp({
  email: 'user@example.com',
  password: 'secret123',
  name: 'User'
})

if (error) console.log(error.message)
else console.log(data.user, data.session)
```

## Login com senha

```typescript
const { data, error } = await client.auth.signIn.withPassword({
  email: 'user@example.com',
  password: 'secret123'
})
```

## Magic link

```typescript
await client.auth.signIn.withMagicLink({
  email: 'user@example.com',
  redirectTo: 'https://app.example.com/callback'
})

const { data, error } = await client.auth.verifyOtp({
  token: 'token-from-url',
  type: 'magic_link'
})
```

## OAuth

```typescript
client.auth.signIn.withOAuth({
  provider: 'github',
  redirectTo: 'https://app.example.com/callback'
})

const { data, error } = client.auth.handleOAuthCallback()
```

## Sessao

```typescript
const session = client.auth.getSession()
const { data } = await client.auth.getUser()

await client.auth.updateUser({
  name: 'New Name',
  metadata: { plan: 'pro' }
})

await client.auth.signOut()
```

## Eventos

```typescript
const { unsubscribe } = client.auth.onAuthStateChange((event, session) => {
  console.log(event, session?.user?.email)
})

unsubscribe()
```

## Referencia

| Metodo | Descricao |
|--------|-----------|
| `signUp({ email, password, name? })` | Cria usuario e sessao. |
| `signIn.withPassword({ email, password })` | Login por senha. |
| `signIn.withMagicLink({ email, redirectTo? })` | Envia magic link. |
| `signIn.withOAuth({ provider, redirectTo? })` | Login OAuth. |
| `verifyOtp({ token, type })` | Verifica token. |
| `signOut()` | Encerra sessao. |
| `getUser()` | Busca usuario atual. |
| `updateUser(updates)` | Atualiza perfil. |
| `getSession()` | Retorna sessao local. |
| `handleOAuthCallback()` | Processa retorno OAuth. |
| `refreshSession(token?)` | Renova access token. |
| `onAuthStateChange(listener)` | Observa eventos de auth. |
