title: "Storage & CDN"
description: "Gerencie buckets, arquivos públicos, arquivos privados e uploads multipart pelo SDK."
---

<Note>
Use storage pelo backend com `NivoAPI` e API key da conta. Buckets públicos geram URL CDN direta. Buckets privados só podem ser lidos pelo backend, SDK ou API key.
</Note>

## Status e uso

```typescript
const status = await nivo.storage.status()
const stats = await nivo.storage.stats()

console.log(status.available)
console.log(stats.usedBytes, stats.totalFiles)
```

## Buckets

Crie buckets públicos para assets que podem ser acessados por qualquer pessoa:

```typescript
const publicBucket = await nivo.storage.createBucket({
  name: 'avatars-publicos',
  public: true,
  fileSizeLimit: 5 * 1024 * 1024,
  allowedMimeTypes: ['image/*'],
  cacheControl: 'public, max-age=31536000'
})
```

Crie buckets privados para arquivos que só devem ser lidos via backend/API key:

```typescript
const privateBucket = await nivo.storage.createBucket({
  name: 'documentos-clientes',
  public: false,
  allowedMimeTypes: ['application/pdf', 'image/*']
})
```

Liste buckets:

```typescript
const buckets = await nivo.storage.listBuckets()
```

## Upload

```typescript
const { file } = await nivo.storage.upload(publicBucket.id, './avatar.png', {
  filename: 'avatar.png',
  folder: 'customers/client_123'
})

// Em bucket público, file.url contém a URL CDN direta.
console.log(file.url)
```

Em bucket privado, `file.url` vem como `null`. Isso é esperado: o arquivo não tem URL pública.

## Listar e deletar arquivos

```typescript
const files = await nivo.storage.listFiles(privateBucket.id, {
  search: 'contrato',
  page: 1,
  limit: 50
})

await nivo.storage.deleteFile('file-id')
await nivo.storage.deleteBucket(privateBucket.id)
```

## Download privado

Arquivos privados são baixados com autenticação pelo SDK/API key.

```typescript
const buffer = await nivo.storage.downloadFile('file-id')
```

Se você precisa mostrar o arquivo em um site, faça isso pelo seu backend:

1. Autentique o usuário da sua aplicação.
2. Verifique se ele pode acessar o arquivo.
3. Use `downloadFile(fileId)` no servidor.
4. Responda o conteúdo com o `Content-Type` correto.

<Warning>
Não coloque API key no browser para acessar arquivo privado. Bucket privado não tem URL pública.
</Warning>

## Multipart upload

Use multipart para arquivos grandes.

```typescript
const multipart = await nivo.storage.buckets.startMultipart(privateBucket.id, {
  filename: 'backup.zip',
  mimeType: 'application/zip',
  totalBytes: fileSize
})

const part = await nivo.storage.buckets.uploadPart(
  privateBucket.id,
  multipart.fileId,
  multipart.uploadId,
  1,
  chunk,
  'part-1'
)

await nivo.storage.buckets.completeMultipart(
  privateBucket.id,
  multipart.fileId,
  multipart.uploadId,
  [part]
)
```

## CDN separado

O recurso `storage.cdn` continua disponível para arquivos públicos globais da CDN.

```typescript
const { file } = await nivo.storage.cdn.upload('./logo.svg', {
  filename: 'logo.svg',
  folder: 'brand'
})

const files = await nivo.storage.cdn.files()
const stats = await nivo.storage.cdn.stats()

await nivo.storage.cdn.delete(file.id)
```

## Referência

| Método | Descrição |
|--------|-----------|
| `storage.status()` | Checa disponibilidade dos serviços de storage. |
| `storage.stats()` | Uso geral de storage. |
| `storage.listBuckets()` | Lista buckets. |
| `storage.createBucket(opts)` | Cria bucket. |
| `storage.getBucket(id)` | Busca bucket. |
| `storage.updateBucket(id, opts)` | Atualiza configuração. |
| `storage.deleteBucket(id)` | Deleta bucket. |
| `storage.listFiles(bucketId, opts)` | Lista arquivos. |
| `storage.upload(bucketId, file, opts)` | Faz upload. |
| `storage.downloadFile(id)` | Baixa arquivo privado usando API key. |
| `storage.deleteFile(id)` | Deleta arquivo. |
| `storage.buckets.startMultipart(id, opts)` | Inicia multipart. |
| `storage.buckets.uploadPart(...)` | Envia parte multipart. |
| `storage.buckets.completeMultipart(...)` | Finaliza multipart. |
| `storage.cdn.upload(file, opts)` | Upload para CDN pública. |
| `storage.cdn.files()` | Lista CDN. |
| `storage.cdn.delete(id)` | Remove arquivo CDN. |
