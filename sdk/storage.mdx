<Note>
  Operações de leitura (listar, buscar) funcionam com `nivo_pub_`. Upload, criação e exclusão exigem Secret Key.
</Note>

## Buckets

Buckets são contêineres de arquivos com regras de acesso, tipos MIME e limite de tamanho próprios.

```typescript
// Listar buckets
const buckets = await nivo.storage.buckets.list()

// Criar bucket público
const bucket = await nivo.storage.buckets.create({
  name: 'avatares',
  public: true,
  sizeLimit: 5 * 1024 * 1024,   // 5 MB
  allowedMime: ['image/*']
})
```

| Parâmetro | Descrição |
|-----------|-----------|
| `name` | Nome único do bucket no projeto. |
| `public` | Se `true`, arquivos são acessíveis por URL pública sem autenticação. |
| `sizeLimit` | Tamanho máximo do arquivo em bytes. |
| `allowedMime` | Tipos MIME aceitos. Ex: `["image/*"]`. Omita para aceitar qualquer tipo. |

## Upload de arquivos

```typescript
const { file } = await nivo.storage.buckets.upload('bucket-id', './imagem.png', {
  filename: 'avatar.png'
})

console.log(file.url)  // URL pública (se bucket for público)
console.log(file.id)   // ID do arquivo para operações futuras
```

## Listar e excluir arquivos

```typescript
// Listar arquivos do bucket
const arquivos = await nivo.storage.buckets.files('bucket-id')

// Excluir arquivo
await nivo.storage.files.delete('file-id')

// URL temporária assinada (buckets privados)
const { url } = await nivo.storage.files.signedUrl('file-id', 3600) // 1 hora
```

## CDN

O CDN é um storage global para assets estáticos com URLs de entrega rápida. Ideal para imagens, fontes e arquivos públicos permanentes.

```typescript
// Upload para CDN
const { file } = await nivo.storage.cdn.upload('./logo.svg')
console.log(file.url) // URL com cache global

// Listar arquivos no CDN
const arquivos = await nivo.storage.cdn.files()

// Excluir do CDN
await nivo.storage.cdn.delete('file-id')
```

## Referência completa

| Método | Descrição |
|--------|-----------|
| `nivo.storage.buckets.list()` | Lista todos os buckets do projeto. |
| `nivo.storage.buckets.create(opts)` | Cria um novo bucket. |
| `nivo.storage.buckets.upload(id, arquivo, opts)` | Faz upload de um arquivo para o bucket. |
| `nivo.storage.buckets.files(id)` | Lista arquivos do bucket. |
| `nivo.storage.buckets.delete(id)` | Exclui o bucket e todos os seus arquivos. |
| `nivo.storage.files.delete(id)` | Exclui um arquivo por ID. |
| `nivo.storage.files.signedUrl(id, ttl)` | Gera URL assinada para acesso temporário. |
| `nivo.storage.cdn.upload(arquivo)` | Faz upload para o CDN. |
| `nivo.storage.cdn.files()` | Lista arquivos no CDN. |
| `nivo.storage.cdn.delete(id)` | Remove arquivo do CDN. |
