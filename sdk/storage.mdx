## Status e uso

```typescript
const status = await nivo.storage.status()
const stats = await nivo.storage.stats()

console.log(status.available)           // true | false
console.log(stats.usedBytes)            // bytes usados
console.log(stats.totalFiles)           // total de arquivos
```

## Buckets

<Tabs>
  <Tab title="Bucket público (CDN)">
    Arquivos públicos são servidos diretamente pela CDN com URL pública.

    ```typescript
    const bucket = await nivo.storage.createBucket({
      name: 'avatares',
      public: true,
      fileSizeLimit: 5 * 1024 * 1024,          // 5 MB
      allowedMimeTypes: ['image/*'],
      cacheControl: 'public, max-age=31536000'  // 1 ano
    })
    ```
  </Tab>
  <Tab title="Bucket privado">
    Arquivos privados só podem ser baixados pela sua API com autenticação.

    ```typescript
    const bucket = await nivo.storage.createBucket({
      name: 'documentos-clientes',
      public: false,
      allowedMimeTypes: ['application/pdf', 'image/*']
    })
    ```
  </Tab>
</Tabs>

### Gerenciar buckets

```typescript
// Listar
const buckets = await nivo.storage.listBuckets()

// Buscar
const bucket = await nivo.storage.getBucket('bucket_id')

// Atualizar
await nivo.storage.updateBucket('bucket_id', {
  fileSizeLimit: 10 * 1024 * 1024  // 10 MB
})

// Deletar (precisa estar vazio)
await nivo.storage.deleteBucket('bucket_id')
```

## Upload

<Tabs>
  <Tab title="Arquivo local">
    ```typescript
    const { file } = await nivo.storage.upload('bucket_id', './foto.jpg', {
      filename: 'foto.jpg',
      folder: 'customers/user_123'
    })

    // Bucket público: URL da CDN
    console.log(file.url)
    // https://cdn.nivo.lat/seu-projeto/avatares/customers/user_123/foto.jpg
    ```
  </Tab>
  <Tab title="Buffer / Stream">
    ```typescript
    import { readFileSync } from 'fs'

    const buffer = readFileSync('./relatorio.pdf')

    const { file } = await nivo.storage.upload('bucket_id', buffer, {
      filename: 'relatorio-jan-2024.pdf',
      mimeType: 'application/pdf',
      folder: 'relatorios/2024'
    })
    ```
  </Tab>
  <Tab title="Express (req.file)">
    ```typescript
    import express from 'express'
    import multer from 'multer'

    const upload = multer({ storage: multer.memoryStorage() })
    const app = express()

    app.post('/upload', upload.single('file'), async (req, res) => {
      const { file: nivoFile } = await nivo.storage.upload(
        'bucket_id',
        req.file!.buffer,
        {
          filename: req.file!.originalname,
          mimeType: req.file!.mimetype,
          folder: `users/${req.user.id}`
        }
      )
      res.json({ url: nivoFile.url, id: nivoFile.id })
    })
    ```
  </Tab>
</Tabs>

## Download privado

Arquivos em buckets privados não têm URL pública. Baixe pelo backend e entregue ao usuário:

```typescript
// No seu endpoint de API
app.get('/arquivos/:fileId', autenticar, async (req, res) => {
  // 1. Verifique se o usuário tem permissão
  const arquivo = await db.arquivos.findById(req.params.fileId)
  if (arquivo.userId !== req.user.id) {
    return res.status(403).json({ error: 'Sem permissão' })
  }

  // 2. Baixe pelo SDK no servidor
  const buffer = await nivo.storage.downloadFile(req.params.fileId)

  // 3. Entregue com o Content-Type correto
  res.set('Content-Type', arquivo.mimeType)
  res.set('Content-Disposition', `attachment; filename="${arquivo.nome}"`)
  res.send(buffer)
})
```

<Warning>
  Nunca gere URLs diretas para arquivos privados. O download deve sempre passar pela sua API.
</Warning>

```typescript
// Listar com filtros
```

## Multipart upload

Use para arquivos grandes (acima de 50 MB):

<Steps>
  <Step title="Iniciar">
    ```typescript
    const multipart = await nivo.storage.buckets.startMultipart('bucket_id', {
      filename: 'backup-2024.zip',
      mimeType: 'application/zip',
      totalBytes: fileSize
    })
    // multipart.fileId, multipart.uploadId
    ```
  </Step>
  <Step title="Enviar partes">
    ```typescript
    const chunkSize = 10 * 1024 * 1024  // 10 MB por parte
    const parts = []

    for (let i = 0; i * chunkSize < fileBuffer.length; i++) {
      const chunk = fileBuffer.slice(i * chunkSize, (i + 1) * chunkSize)
      const part = await nivo.storage.buckets.uploadPart(
        'bucket_id',
        multipart.fileId,
        multipart.uploadId,
        i + 1,   // número da parte (começa em 1)
        chunk,
        `part-${i + 1}`
      )
      parts.push(part)
    }
    ```
  </Step>
  <Step title="Finalizar">
    ```typescript
    const file = await nivo.storage.buckets.completeMultipart(
      'bucket_id',
      multipart.fileId,
      multipart.uploadId,
      parts
    )

    console.log(file.url)
    ```
  </Step>
</Steps>

## CDN público separado

Para assets globais (imagens, fontes, scripts) sem estrutura de bucket:

```typescript
// Upload direto para CDN
const { file } = await nivo.storage.cdn.upload('./logo.svg', {
  filename: 'logo.svg',
  folder: 'brand'
})
console.log(file.url)

// Listar
const cdnFiles = await nivo.storage.cdn.files()

// Estatísticas
const cdnStats = await nivo.storage.cdn.stats()
console.log(cdnStats.usedBytes, cdnStats.totalFiles)

// Deletar
await nivo.storage.cdn.delete(file.id)
```

## Referência completa

| Método | Descrição |
|--------|-----------|
| `storage.status()` | Verifica disponibilidade do storage. |
| `storage.stats()` | Uso geral (bytes e arquivos). |
| `storage.listBuckets()` | Lista buckets. |
| `storage.createBucket(opts)` | Cria bucket. |
| `storage.getBucket(id)` | Busca bucket por ID. |
| `storage.updateBucket(id, opts)` | Atualiza configurações. |
| `storage.deleteBucket(id)` | Deleta bucket (precisa estar vazio). |
| `storage.listFiles(bucketId, opts)` | Lista arquivos com filtros. |
| `storage.upload(bucketId, file, opts)` | Faz upload de arquivo. |
| `storage.downloadFile(id)` | Baixa arquivo privado. |
| `storage.deleteFile(id)` | Deleta arquivo. |
| `storage.buckets.startMultipart(id, opts)` | Inicia upload multipart. |
| `storage.buckets.uploadPart(...)` | Envia parte do multipart. |
| `storage.buckets.completeMultipart(...)` | Finaliza multipart. |
| `storage.cdn.upload(file, opts)` | Upload para CDN pública. |
| `storage.cdn.files()` | Lista arquivos da CDN. |
| `storage.cdn.stats()` | Estatísticas da CDN. |
| `storage.cdn.delete(id)` | Remove arquivo da CDN. |
