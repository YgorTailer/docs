---
title: "Arquivos do App"
description: "Leia, escreva e liste arquivos dentro do container em execução pelo SDK."
---

<Warning>
  Este recurso lê e escreve arquivos **dentro do container em execução**. Os dados são perdidos quando o app é redeploy. Use [Storage & CDN](/sdk/storage) para persistência.
</Warning>

## Listar arquivos

```typescript
const files = await nivo.files.list('app_id', '/app/logs')
// Retorna: [{ name, path, size, isDirectory, modifiedAt }]
```

## Ler arquivo

```typescript
const content = await nivo.files.read('app_id', '/app/config.json')
// Retorna: string com o conteúdo do arquivo
```

## Escrever arquivo

```typescript
await nivo.files.write('app_id', '/app/config.json', JSON.stringify({
  debug: true,
  logLevel: 'verbose'
}, null, 2))
```

## Deletar arquivo

```typescript
await nivo.files.delete('app_id', '/app/logs/old.log')
```

## Casos de uso

<AccordionGroup>
  <Accordion title="Ler logs do app">
    ```typescript
    const logs = await nivo.files.read('app_id', '/app/logs/app.log')
    const lines = logs.split('\n').slice(-100)  // últimas 100 linhas
    ```
  </Accordion>
  <Accordion title="Atualizar config em runtime">
    ```typescript
    const config = JSON.parse(
      await nivo.files.read('app_id', '/app/config.json')
    )

    config.featureFlags.newCheckout = true

    await nivo.files.write(
      'app_id',
      '/app/config.json',
      JSON.stringify(config, null, 2)
    )
    ```
  </Accordion>
  <Accordion title="Listar diretório">
    ```typescript
    const entries = await nivo.files.list('app_id', '/app/uploads')

    for (const entry of entries) {
      console.log(`${entry.isDirectory ? '📁' : '📄'} ${entry.name} (${entry.size} bytes)`)
    }
    ```
  </Accordion>
</AccordionGroup>

## Referência

| Método | Descrição |
|--------|-----------|
| `files.list(appId, path)` | Lista arquivos e diretórios. |
| `files.read(appId, path)` | Lê conteúdo de um arquivo. |
| `files.write(appId, path, content)` | Escreve/sobrescreve arquivo. |
| `files.delete(appId, path)` | Remove arquivo. |
