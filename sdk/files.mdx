---
title: "Arquivos do App"
description: "Leia, escreva e remova arquivos dentro do container em execucao."
---

<Warning>
  O app precisa estar em execucao. Mudancas feitas no filesystem do container podem ser perdidas no proximo deploy. Para configuracao permanente, prefira `updateEnv()` ou inclua o arquivo no ZIP.
</Warning>

## Listar arquivos

```typescript
const { files } = await nivo.apps.listFiles('app-id', '/app')

for (const file of files) {
  console.log(file.name, file.type, file.size)
}
```

## Ler arquivo

```typescript
const { content } = await nivo.apps.readFile('app-id', '/app/package.json')
console.log(content)
```

## Escrever arquivo

```typescript
await nivo.apps.writeFile(
  'app-id',
  '/app/runtime-config.json',
  JSON.stringify({ plan: 'pro' })
)
```

## Deletar arquivo

```typescript
await nivo.apps.deleteFile('app-id', '/app/tmp/cache.json')
```

## Referencia

| Metodo | Descricao |
|--------|-----------|
| `apps.listFiles(id, path?)` | Lista arquivos. Padrao: `/app`. |
| `apps.readFile(id, path)` | Le arquivo como string. |
| `apps.writeFile(id, path, content)` | Cria ou sobrescreve arquivo. |
| `apps.deleteFile(id, path)` | Remove arquivo. |
