## Criar banco

```typescript
const db = await nivo.databases.create({
  name: 'cliente_123',
  password: process.env.DB_PASSWORD!,
  engine: 'postgresql',
  ramMb: 512
})

console.log(db.connectionString)
```

## Listar e buscar

```typescript
const databases = await nivo.databases.list()
const db = await nivo.databases.get('db-id')
```

## Ciclo de vida

```typescript
await nivo.databases.start('db-id')
await nivo.databases.stop('db-id')
await nivo.databases.restart('db-id')
```

## Atualizar configuracao

```typescript
await nivo.databases.update('db-id', {
  ramMb: 1024
})

await nivo.databases.updateRam('db-id', 1024)
```

## Status e logs

```typescript
const stats = await nivo.databases.stats('db-id')
const logs = await nivo.databases.logs('db-id', { tail: 300 })

console.log(stats.memoryUsageMB, stats.cpuUsagePct)
console.log(logs.log)
```

## Query SQL

Disponivel para PostgreSQL.

```typescript
const result = await nivo.databases.query(
  'db-id',
  'SELECT * FROM customers LIMIT 10'
)

console.log(result.columns)
console.log(result.rows)
```

## Tabelas e CRUD

```typescript
const tables = await nivo.databases.tables('db-id')

const rows = await nivo.databases.tableRows('db-id', 'customers', {
  page: 1,
  limit: 50
})

await nivo.databases.insertRow('db-id', 'customers', {
  name: 'Cliente exemplo'
})
```

## Deletar banco

```typescript
await nivo.databases.delete('db-id')
```

## Referencia

| Metodo | Descricao |
|--------|-----------|
| `list()` | Lista bancos. |
| `get(id)` | Retorna banco e connection string. |
| `create(opts)` | Cria banco. |
| `update(id, opts)` | Atualiza configuracao. |
| `updateRam(id, ramMb)` | Atalho para alterar RAM. |
| `delete(id)` | Remove banco. |
| `start(id)` | Inicia banco. |
| `stop(id)` | Para banco. |
| `restart(id)` | Reinicia banco. |
| `stats(id)` | CPU, memoria, rede, uptime e imagem. |
| `logs(id, { tail })` | Logs do container. |
| `query(id, sql)` | Executa SQL em PostgreSQL. |
| `tables(id)` | Lista tabelas. |
| `tableRows(id, table, opts)` | Lista linhas. |
| `insertRow(id, table, data)` | Insere linha. |
| `updateRow(id, table, pkValue, opts)` | Atualiza valor. |
| `deleteRows(id, table, opts)` | Remove linhas. |
