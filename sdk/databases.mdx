## Listar e buscar

```typescript
// Listar todos os bancos do projeto
const databases = await nivo.databases.list()

// Buscar por ID
const db = await nivo.databases.get('db_id')

// Status atual
console.log(db.status) // 'running' | 'stopped' | 'starting' | 'error'
```

---

## Ciclo de vida

```typescript
// Parar (para cobrança de compute enquanto parado)
await nivo.databases.stop('db_id')

// Iniciar
await nivo.databases.start('db_id')

// Reiniciar
await nivo.databases.restart('db_id')
```

## Queries SQL

Execute queries diretamente pelo SDK (apenas PostgreSQL e MySQL):

<Tabs>
  <Tab title="SELECT">
    ```typescript
    const result = await nivo.databases.query('db_id', {
      sql: 'SELECT * FROM usuarios WHERE ativo = $1 LIMIT 10',
      params: [true]
    })

    console.log(result.rows)
    console.log(`${result.rowCount} linhas retornadas`)
    ```
  </Tab>
  <Tab title="INSERT">
    ```typescript
    const result = await nivo.databases.query('db_id', {
      sql: 'INSERT INTO pedidos (usuario_id, total) VALUES ($1, $2) RETURNING id',
      params: ['user_123', 99.90]
    })

    console.log(result.rows[0].id)
    ```
  </Tab>
  <Tab title="Transação">
    ```typescript
    await nivo.databases.query('db_id', {
      sql: 'BEGIN'
    })

    try {
      await nivo.databases.query('db_id', {
        sql: 'UPDATE contas SET saldo = saldo - $1 WHERE id = $2',
        params: [100, 'conta_origem']
      })
      await nivo.databases.query('db_id', {
        sql: 'UPDATE contas SET saldo = saldo + $1 WHERE id = $2',
        params: [100, 'conta_destino']
      })
      await nivo.databases.query('db_id', { sql: 'COMMIT' })
    } catch (err) {
      await nivo.databases.query('db_id', { sql: 'ROLLBACK' })
      throw err
    }
    ```
  </Tab>
</Tabs>

<Note>
  Queries SQL estão sujeitas ao rate limit de 60 req/min.
</Note>

## Métricas e estatísticas

```typescript
const stats = await nivo.databases.stats('db_id')

console.log(`CPU: ${stats.cpuPercent}%`)
console.log(`RAM: ${stats.memMb} MB`)
console.log(`Disk: ${stats.diskMb} MB`)
console.log(`Conexões ativas: ${stats.connections}`)
```

## Backups

```typescript
// Listar backups disponíveis
const backups = await nivo.databases.listBackups('db_id')

// Criar backup manual
const backup = await nivo.databases.createBackup('db_id')

// Restaurar de um backup
await nivo.databases.restoreBackup('db_id', backup.id)
```

<Note>
  Backups automáticos diários estão disponíveis nos planos Pro e Builder.
</Note>

---

## Redimensionar

```typescript
// Alterar tier de compute
await nivo.databases.resize('db_id', {
  computeTier: 'ORBIT'  // mais RAM e vCPU
})
```

---

## Deletar

```typescript
await nivo.databases.delete('db_id')
```

<Warning>
  A deleção é permanente e imediata. Todos os dados são perdidos. Crie um backup antes de deletar.
</Warning>

## Referência completa

| Método | Descrição |
|--------|-----------|
| `databases.list()` | Lista bancos do projeto. |
| `databases.get(id)` | Busca banco por ID. |
| `databases.create(opts)` | Provisiona novo banco. |
| `databases.delete(id)` | Remove banco permanentemente. |
| `databases.start(id)` | Inicia o banco. |
| `databases.stop(id)` | Para o banco. |
| `databases.restart(id)` | Reinicia o banco. |
| `databases.query(id, opts)` | Executa SQL. |
| `databases.stats(id)` | Métricas de CPU, RAM e disco. |
| `databases.resize(id, opts)` | Altera tier de compute. |
| `databases.listBackups(id)` | Lista backups disponíveis. |
| `databases.createBackup(id)` | Cria backup manual. |
| `databases.restoreBackup(id, backupId)` | Restaura de backup. |
