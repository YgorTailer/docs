## Criar banco de dados

Provisiona um novo container de banco de dados. Retorna uma `connectionString` pronta para uso. O status inicial ĂŠ `provisioning` e muda para `running` em segundos.

```typescript
const db = await nivo.databases.create({
  name: 'meudb',
  password: 'senha123',
  engine: 'postgresql',
  ramMb: 512
})

console.log(db.connectionString)
// postgres://meudb:senha123@host:5432/meudb
```

```typescript

```

## Executar query SQL

Executa queries diretamente pelo SDK. **Somente PostgreSQL.** O banco precisa estar com status `running`.

```typescript
const resultado = await nivo.databases.query(
  'db-id',
  'SELECT * FROM usuarios LIMIT 10'
)

console.log(resultado.rows)
console.log(resultado.columns)
```

## Tabelas & CRUD

Inspecione e manipule dados sem escrever SQL. **Somente PostgreSQL.**

```typescript
// Listar tabelas
const tabelas = await nivo.databases.tables('db-id')

// Buscar linhas de uma tabela
const { rows, columns, total } = await nivo.databases.tableRows('db-id', 'usuarios')

// Inserir linha
const criado = await nivo.databases.insertRow('db-id', 'usuarios', {
  email: 'usuario@exemplo.com',
  nome: 'JoĂŁo'
})
```
