## ZIP upload

Para apps com `sourceType: 'zip'`, `commit()` envia um arquivo ZIP e cria um deployment.

```typescript
const { deploymentId } = await nivo.apps.commit(
  'app-id',
  './app.zip',
  'app.zip'
)
```

Tambem aceita `Buffer` e `Readable`.

```typescript
import { readFileSync, createReadStream } from 'node:fs'

await nivo.apps.commit('app-id', readFileSync('./app.zip'), 'app.zip')
await nivo.apps.commit('app-id', createReadStream('./app.zip'), 'app.zip')
```

## Commit e aguardar

```typescript
const result = await nivo.apps.commitAndWait('app-id', './app.zip', 'app.zip', {
  intervalMs: 3000,
  timeoutMs: 30 * 60 * 1000,
  onLog: (chunk) => process.stdout.write(chunk),
  onStatus: (status) => console.log(status)
})

console.log(result.status)
```

## GitHub redeploy

Para apps com `sourceType: 'github'`, use `deploy()` ou `deployAndWait()`.

```typescript
const { deploymentId } = await nivo.apps.deploy('app-id')

await nivo.apps.deployAndWait('app-id', {
  onLog: (chunk) => process.stdout.write(chunk)
})
```

## Fluxo completo para managers

```typescript
import { NivoAPI } from '@nivo-lat/sdk'

const nivo = new NivoAPI(process.env.NIVO_API_KEY!)

const app = await nivo.apps.create({
  name: 'cliente-vendas',
  sourceType: 'zip',
  runtime: 'node20',
  installCmd: 'npm ci',
  buildCmd: 'npm run build',
  startCmd: 'npm start',
  ramMb: 256,
  envVars: {
    CLIENT_ID: 'client_123'
  }
})

await nivo.apps.commitAndWait(app.id, './release.zip', 'release.zip', {
  onLog: (chunk) => process.stdout.write(chunk)
})

await nivo.apps.updateEnv(app.id, {
  PLAN: 'enterprise'
})
```

## Status final

`waitForDeployment()`, `commitAndWait()` e `deployAndWait()` retornam os logs do deployment com status final.

```typescript
const logs = await nivo.apps.waitForDeployment('app-id', 'deployment-id')

if (logs.status !== 'success') {
  throw new Error(`Deploy falhou: ${logs.status}`)
}
```
