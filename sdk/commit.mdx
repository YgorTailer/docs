---
title: "Commit & Deploy"
description: "Envie um arquivo .zip com o código da sua aplicação e acione um redeploy automaticamente."
---

O deploy via SDK funciona em 3 etapas: criar o app, enviar o código (commit) e acompanhar o build.

## 1. Criar o app

Faça isso uma vez. O `id` retornado é usado em todas as operações seguintes.

```typescript
const app = await nivo.apps.create({
  name: 'meu-api',
  runtime: 'node20',
  startCmd: 'node dist/index.js',
  buildCmd: 'npm run build',
  installCmd: 'npm ci',
  ramMb: 256
})

console.log(app.id) // salve este ID
```

| Parâmetro | Obrigatório | Descrição |
|-----------|-------------|-----------|
| `name` | Sim | Nome do app (1–48 caracteres). |
| `runtime` | Não | Runtime. Ex: `node20`, `python3.11`. Padrão: `node20`. |
| `startCmd` | Não | Comando de inicialização. Ex: `node dist/index.js`. |
| `buildCmd` | Não | Comando de build. Ex: `npm run build`. |
| `installCmd` | Não | Comando de instalação. Ex: `npm ci`. |
| `buildSystem` | Não | `nivopack` (padrão), `nixpacks` ou `dockerfile`. |
| `ramMb` | Não | RAM em MB. Padrão: `256`. |
| `type` | Não | `site` (padrão) ou `worker`. |

## 2. Enviar o código (commit)

Use `nivo.apps.commit()` para enviar o zip e acionar o build. O app continua rodando enquanto o novo build é processado.

<CodeGroup>
```typescript Buffer
import { readFileSync } from 'fs'

const zip = readFileSync('./dist.zip')

const { deploymentId } = await nivo.apps.commit({
  id: app.id,
  file: zip,
  filename: 'app.zip'
})
```

```typescript Stream
import { createReadStream } from 'fs'

const { deploymentId } = await nivo.apps.commit({
  id: app.id,
  file: createReadStream('./dist.zip'),
  filename: 'app.zip'
})
```

```typescript Caminho do arquivo
const { deploymentId } = await nivo.apps.commit({
  id: app.id,
  file: './dist.zip'
})
```
</CodeGroup>

## 3. Aguardar o build

O `commit()` é assíncrono. Use `deploymentLogs()` para acompanhar o progresso.

```typescript
let concluido = false

while (!concluido) {
  await new Promise(r => setTimeout(r, 2000))

  const { log } = await nivo.apps.deploymentLogs(app.id, deploymentId)
  const ultimo = log.at(-1)

  if (ultimo?.status === 'success') {
    console.log('✅ Deploy concluído!')
    concluido = true
  } else if (ultimo?.status === 'failed') {
    console.log('❌ Deploy falhou:', ultimo)
    concluido = true
  }
}
```

## Fluxo completo

```typescript
import { readFileSync } from 'fs'
import { NivoAPI } from '@nivocloud/sdk'

const nivo = new NivoAPI(process.env.NIVO_API_KEY!)

// 1. Criar app
const app = await nivo.apps.create({
  name: 'meu-api',
  runtime: 'node20',
  startCmd: 'node dist/index.js',
  buildCmd: 'npm run build',
  installCmd: 'npm ci'
})

// 2. Enviar código
const zip = readFileSync('./dist.zip')
const { deploymentId } = await nivo.apps.commit({ id: app.id, file: zip })
console.log('Build iniciado:', deploymentId)

// 3. Aguardar conclusão
let concluido = false
while (!concluido) {
  await new Promise(r => setTimeout(r, 2000))
  const { log } = await nivo.apps.deploymentLogs(app.id, deploymentId)
  const ultimo = log.at(-1)
  if (ultimo?.status === 'success') { console.log('✅ Sucesso!'); concluido = true }
  else if (ultimo?.status === 'failed') { console.log('❌ Falhou'); concluido = true }
}
```
