## Criar app

```typescript
const app = await nivo.apps.create({
  name: 'sales-bot',
  sourceType: 'zip',
  buildSystem: 'nivopack',
  runtime: 'node20',
  startCmd: 'npm start',
  installCmd: 'npm ci',
  buildCmd: 'npm run build',
  ramMb: 256,
  envVars: {
    CUSTOMER_ID: 'cus_123'
  }
})
```

Para GitHub:

```typescript
const app = await nivo.apps.create({
  name: 'sales-bot',
  sourceType: 'github',
  repoFullName: 'empresa/sales-bot',
  repoBranch: 'main',
  buildSystem: 'nivopack',
  runtime: 'node20',
  startCmd: 'npm start'
})
```

Para Dockerfile:

```typescript
const app = await nivo.apps.create({
  name: 'sales-bot',
  sourceType: 'github',
  repoFullName: 'empresa/sales-bot',
  repoBranch: 'main',
  buildSystem: 'dockerfile',
  dockerfilePath: 'Dockerfile',
  startCmd: 'npm start'
})
```

Para imagem Docker pública:

```typescript
const app = await nivo.apps.create({
  name: 'sales-bot',
  sourceType: 'image',
  dockerImage: 'ghcr.io/empresa/sales-bot:latest',
  type: 'site',
  startCmd: 'npm start',
  ramMb: 256
})
```

## Ciclo de vida

```typescript
await nivo.apps.start(app.id)
await nivo.apps.stop(app.id)
await nivo.apps.restart(app.id)
```

## Status e logs

```typescript
const app = await nivo.apps.get('app-id')
const stats = await nivo.apps.stats('app-id')
const logs = await nivo.apps.logs('app-id', { tail: 300 })

console.log(app.status)
console.log(stats.memUsageMb, stats.cpuPct)
console.log(logs.log)
```

## Atualizar configuracao

```typescript
await nivo.apps.update('app-id', {
  runtime: 'node20',
  buildSystem: 'dockerfile',
  dockerfilePath: 'Dockerfile',
  startCmd: 'npm start',
  buildCmd: 'npm run build',
  ramMb: 512,
  autoRestart: true
})

await nivo.apps.updateRam('app-id', 512)
```

## Variaveis de ambiente

`updateEnv()` faz merge: adiciona ou altera somente as chaves enviadas.

```typescript
await nivo.apps.updateEnv('app-id', {
  STRIPE_KEY: 'sk_live_...',
  PLAN: 'pro'
})
```

`replaceEnv()` substitui tudo de forma explicita.

```typescript
await nivo.apps.replaceEnv('app-id', {
  NODE_ENV: 'production'
})
```

Para remover chaves:

```typescript
await nivo.apps.removeEnv('app-id', ['OLD_KEY'])
```

## Deployments

```typescript
const deployments = await nivo.apps.deployments('app-id')
const logs = await nivo.apps.deploymentLogs('app-id', 'deployment-id')

await nivo.apps.deployAndWait('app-id', {
  onLog: (chunk) => process.stdout.write(chunk)
})
```

## Deletar app

```typescript
await nivo.apps.delete('app-id')
```

## Referencia

| Metodo | Descricao |
|--------|-----------|
| `list()` | Lista apps. |
| `get(id)` | Retorna app e status. |
| `create(opts)` | Cria app ZIP, GitHub ou imagem Docker. |
| `update(id, opts)` | Atualiza configuracao. |
| `updateRam(id, ramMb)` | Atalho para alterar RAM. |
| `delete(id)` | Remove app e recursos. |
| `upload(id, file)` | Envia ZIP e cria deployment. |
| `uploadAndWait(id, file)` | Envia ZIP e espera terminar. |
| `commit(id, file)` | Alias profissional para upload de codigo. |
| `commitAndWait(id, file)` | Upload de codigo e espera deployment. |
| `deploy(id)` | Redeploy do ultimo codigo ou repo GitHub. |
| `deployAndWait(id)` | Redeploy e espera terminar. |
| `start(id)` | Inicia app. |
| `stop(id)` | Para app. |
| `restart(id)` | Reinicia app. |
| `logs(id, { tail })` | Logs de runtime. |
| `stats(id)` | CPU, memoria, rede, uptime e imagem. |
| `getEnv(id)` | Lista env vars. |
| `updateEnv(id, vars)` | Merge de env vars. |
| `replaceEnv(id, vars)` | Substitui env vars. |
| `removeEnv(id, keys)` | Remove env vars. |
