## Listar apps

```typescript
const apps = await nivo.apps.list()
// App[]
```

## Buscar app

```typescript
const app = await nivo.apps.get('app_id')
```

## Criar app

```typescript
const app = await nivo.apps.create({
  name: 'minha-api',
  type: 'site',          // 'site' | 'worker'
  runtime: 'node20',
  startCmd: 'npm start',
  buildCmd: 'npm run build',
  installCmd: 'npm install',
  ramMb: 512
})
```

### Tipos de app

<Tabs>
  <Tab title="Site (HTTP)">
    Recebe tráfego HTTP/HTTPS. Recebe uma URL pública automaticamente.

    ```typescript
    const app = await nivo.apps.create({
      name: 'minha-api',
      type: 'site',
      runtime: 'node20',
      port: 3000,
      startCmd: 'node dist/index.js'
    })
    console.log(app.url) // https://minha-api.nivo.lat
    ```
  </Tab>
  <Tab title="Worker (background)">
    Roda em background sem porta HTTP. Ideal para filas, cron jobs e processamento.

    ```typescript
    const worker = await nivo.apps.create({
      name: 'worker-emails',
      type: 'worker',
      runtime: 'node20',
      startCmd: 'node worker.js'
    })
    ```
  </Tab>
</Tabs>

## Variáveis de ambiente

```typescript
// Definir variáveis
await nivo.apps.setEnvVars('app_id', {
  DATABASE_URL: 'postgres://...',
  REDIS_URL: 'redis://...',
  SECRET_KEY: 'super-secreto'
})

// Listar variáveis (valores mascarados)
const envVars = await nivo.apps.getEnvVars('app_id')
```

<Warning>
  Após alterar variáveis de ambiente, um novo deploy é necessário para que as mudanças entrem em vigor.
</Warning>

## Ciclo de vida

```typescript
// Parar
await nivo.apps.stop('app_id')

// Iniciar
await nivo.apps.start('app_id')

// Reiniciar
await nivo.apps.restart('app_id')
```

---

## Atualizar configuração

```typescript
await nivo.apps.update('app_id', {
  ramMb: 1024,
  startCmd: 'node dist/server.js'
})
```

---

## Domínio personalizado

```typescript
// Adicionar domínio
const result = await nivo.apps.addDomain('app_id', 'meusite.com.br')

// Verificar DNS (retorna instruções)
console.log(result.instructions)

// Remover domínio
await nivo.apps.removeDomain('app_id', 'meusite.com.br')
```

---

## Logs e métricas

```typescript
// Logs de runtime
const logs = await nivo.apps.logs('app_id', { tail: 200 })

// Métricas (CPU, RAM, uptime)
const metrics = await nivo.apps.metrics('app_id')
console.log(`CPU: ${metrics.cpuPercent}%`)
console.log(`RAM: ${metrics.memMb} MB`)
```

---

## Deletar app

```typescript
await nivo.apps.delete('app_id')
```

<Warning>
  A deleção é permanente. O app, seus logs e deployments são removidos imediatamente.
</Warning>

## Referência completa

| Método | Descrição |
|--------|-----------|
| `apps.list()` | Lista todos os apps do projeto. |
| `apps.get(id)` | Busca um app por ID. |
| `apps.create(opts)` | Cria um novo app. |
| `apps.update(id, opts)` | Atualiza configurações do app. |
| `apps.delete(id)` | Remove o app permanentemente. |
| `apps.start(id)` | Inicia o app. |
| `apps.stop(id)` | Para o app. |
| `apps.restart(id)` | Reinicia o app. |
| `apps.setEnvVars(id, vars)` | Define variáveis de ambiente. |
| `apps.getEnvVars(id)` | Lista variáveis de ambiente. |
| `apps.logs(id, opts)` | Retorna logs do container. |
| `apps.metrics(id)` | Retorna métricas de CPU e RAM. |
| `apps.addDomain(id, domain)` | Adiciona domínio personalizado. |
| `apps.removeDomain(id, domain)` | Remove domínio personalizado. |
