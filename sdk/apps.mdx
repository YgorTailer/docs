title: "Gerenciar Apps"
description: "Controle o ciclo de vida dos seus apps: listar, iniciar, parar, ver logs, deployments e variáveis de ambiente."
---

## Listar e buscar apps

```typescript
// Listar todos os apps do projeto
const apps = await nivo.apps.list()

// Buscar um app específico
const app = await nivo.apps.get('app-id')
```

## Iniciar e parar

```typescript
await nivo.apps.start('app-id')
await nivo.apps.stop('app-id')
```

## Logs em tempo real

Retorna os logs `stdout`/`stderr` do container em execução.

```typescript
const { log } = await nivo.apps.logs('app-id')
console.log(log)
```

## Deployments

```typescript
// Listar histórico de deployments
const deployments = await nivo.apps.deployments('app-id')

// Logs de um deployment específico
const { log } = await nivo.apps.deploymentLogs('app-id', 'deployment-id')

// Acionar redeploy sem enviar código novo (usa o último commit)
await nivo.apps.deploy('app-id')
```

## Variáveis de ambiente

Atualiza as variáveis de ambiente do app. Esta operação **substitui todas** as variáveis existentes.

```typescript
await nivo.apps.setEnv('app-id', {
  NODE_ENV: 'production',
  PORT: '3000',
  DATABASE_URL: 'postgres://...'
})
```

<Warning>
  `setEnv()` é uma substituição completa. Inclua **todas** as variáveis que quiser manter, não só as novas.
</Warning>

## Referência completa

| Método | Descrição |
|--------|-----------|
| `nivo.apps.list()` | Lista todos os apps do projeto. |
| `nivo.apps.get(id)` | Retorna detalhes de um app específico. |
| `nivo.apps.create(opts)` | Cria um novo app. Veja [Commit & Deploy](/sdk/commit). |
| `nivo.apps.commit(opts)` | Envia zip e aciona redeploy. |
| `nivo.apps.deploy(id)` | Redeploy sem enviar código novo. |
| `nivo.apps.start(id)` | Inicia o container do app. |
| `nivo.apps.stop(id)` | Para o container do app. |
| `nivo.apps.logs(id)` | Retorna logs do container em execução. |
| `nivo.apps.deployments(id)` | Lista os últimos 50 deployments. |
| `nivo.apps.deploymentLogs(id, depId)` | Logs de um deployment específico. |
| `nivo.apps.setEnv(id, vars)` | Substitui todas as variáveis de ambiente. |
