## Instalação

<CodeGroup>
```bash npm
npm install @nivo-lat/sdk
```

```bash yarn
yarn add @nivo-lat/sdk
```

```bash pnpm
pnpm add @nivo-lat/sdk
```

## Cliente oficial

Use `NivoAPI` em ambientes de servidor com uma API key da conta e o `projectId` do projeto que será gerenciado.

```typescript
import { NivoAPI } from '@nivo-lat/sdk'

const nivo = new NivoAPI(process.env.NIVO_API_KEY!, {
  projectId: process.env.NIVO_PROJECT_ID!
})
```

<Warning>
Mantenha `NIVO_API_KEY` em variáveis de ambiente do servidor.
</Warning>

## Operações de infraestrutura

Use `NivoAPI` para gerenciar recursos do projeto informado em `projectId`.

```typescript
const app = await nivo.apps.create({
  name: 'minha-api',
  runtime: 'node20',
  startCmd: 'node dist/server.js',
  buildCmd: 'npm run build',
  installCmd: 'npm ci',
  ramMb: 512,
})

await nivo.apps.commitAndWait(app.id, './build.zip', 'build.zip', {
  onLog: (chunk) => process.stdout.write(chunk),
})

console.log(`App disponível em: https://${app.subdomain}.nivo.lat`)
```

## Deploy via GitHub

```typescript
const app = await nivo.apps.create({
  name: 'minha-api',
  sourceType: 'github',
  repoFullName: 'empresa/minha-api',
  repoBranch: 'main',
  runtime: 'node20',
  startCmd: 'node dist/server.js',
})

await nivo.apps.deployAndWait(app.id, {
  onLog: (chunk) => process.stdout.write(chunk),
})
```

## Storage privado

Buckets públicos retornam URL CDN direta. Buckets privados usam download autenticado pelo SDK.

```typescript
const buffer = await nivo.storage.downloadFile('file-id')
```

## Recursos disponíveis

| Recurso | Operações |
|---------|-----------|
| **Apps** | Criar, atualizar, iniciar, parar, reiniciar, deploy, commit, logs e métricas |
| **Arquivos do app** | Listar, ler, escrever e remover arquivos dentro do container |
| **Bancos de dados** | Criar, consultar, iniciar, parar, reiniciar, logs, métricas e tabelas |
| **Storage** | Buckets, upload, multipart upload, download privado autenticado e CDN pública |

## Tratamento de erros

Todos os erros da API retornam uma instância de `NivoError` com status HTTP, mensagem e detalhe opcional.

```typescript
import { NivoAPI, NivoError } from '@nivo-lat/sdk'

const nivo = new NivoAPI(process.env.NIVO_API_KEY!, {
  projectId: process.env.NIVO_PROJECT_ID!
})

try {
  const app = await nivo.apps.get('app-id')
} catch (err) {
  if (err instanceof NivoError) {
    console.error(`[${err.status}] ${err.message}`)
    if (err.detail) console.error(err.detail)
  }
}
```
