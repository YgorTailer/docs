## O que é o SDK?

O `@nivocloud/sdk` é o pacote oficial do Nivo para Node.js e TypeScript. Com ele você gerencia toda a infraestrutura via código — sem precisar do CLI ou do dashboard.

**Casos de uso comuns:**
- Scripts de automação e CI/CD
- Backends que gerenciam infraestrutura dinamicamente
- Ferramentas internas de DevOps

## Instalação

<CodeGroup>
```bash npm
npm install @nivocloud/sdk
```

```bash yarn
yarn add @nivocloud/sdk
```

```bash pnpm
pnpm add @nivocloud/sdk
```
</CodeGroup>

## Exemplo rápido

```typescript
import { NivoAPI } from '@nivocloud/sdk'

const nivo = new NivoAPI(process.env.NIVO_API_KEY!)

// Listar apps do projeto
const apps = await nivo.apps.list()

// Listar bancos de dados
const databases = await nivo.databases.list()

// Ver uso do storage
const stats = await nivo.storage.stats()
```

## Tipos de chave de API

Gere suas chaves em **Dashboard → API Keys**.

| Tipo | Prefixo | Quando usar |
|------|---------|-------------|
| **Secret Key** | `nivo_secret_` | Backend, scripts locais. **Nunca exponha no frontend.** |
| **Publishable Key** | `nivo_pub_` | Somente leitura de storage/CDN. Seguro para uso no browser. |

## Tratamento de erros

O SDK lança `NivoError` em qualquer resposta com erro HTTP.

```typescript
import { NivoAPI, NivoError } from '@nivocloud/sdk'

const nivo = new NivoAPI(process.env.NIVO_API_KEY!)

try {
  const app = await nivo.apps.get('id-invalido')
} catch (err) {
  if (err instanceof NivoError) {
    console.log(err.status)   // ex: 404
    console.log(err.message)  // "App not found"
  }
}
```
