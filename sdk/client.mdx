## Instalação

```bash
npm install @nivocloud/sdk
```

## Criando o cliente

Importe `NivoAPI` e passe sua Secret Key. Gere a chave em **Dashboard → API Keys**.

<CodeGroup>
```typescript TypeScript
import { NivoAPI } from '@nivocloud/sdk'

const nivo = new NivoAPI(process.env.NIVO_API_KEY!)

// Recursos disponíveis:
// nivo.apps       → gerenciar apps
// nivo.databases  → gerenciar bancos de dados
// nivo.storage    → buckets, arquivos e CDN
```

```javascript JavaScript (ESM)
import { NivoAPI } from '@nivocloud/sdk'

const nivo = new NivoAPI(process.env.NIVO_API_KEY)
```

```javascript JavaScript (CJS)
const { NivoAPI } = require('@nivocloud/sdk')

const nivo = new NivoAPI(process.env.NIVO_API_KEY)
```
</CodeGroup>

## Variável de ambiente

Nunca coloque sua Secret Key diretamente no código. Use variáveis de ambiente.

```bash .env
NIVO_API_KEY=nivo_secret_...
```

```typescript
const nivo = new NivoAPI(process.env.NIVO_API_KEY!)
```

## Opções avançadas

```typescript
const nivo = new NivoAPI('nivo_secret_...', {
  baseUrl: 'https://api.nivo.lat/api' // padrão
})
```

## Tratamento de erros

```typescript
import { NivoAPI, NivoError } from '@nivocloud/sdk'

try {
  const app = await nivo.apps.get('app-id')
} catch (err) {
  if (err instanceof NivoError) {
    console.log(err.status)   // código HTTP (ex: 404, 401)
    console.log(err.message)  // mensagem de erro da API
    console.log(err.detail)   // detalhe adicional (opcional)
  }
}
```
