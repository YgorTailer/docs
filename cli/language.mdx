## nivo lang

O CLI suporta português e inglês. O idioma afeta mensagens de ajuda, prompts interativos e saídas do terminal.

### Definir idioma

<Tabs>
  <Tab title="Português">
    ```bash
    nivo lang set pt
    ```
  </Tab>
  <Tab title="English">
    ```bash
    nivo lang set en
    ```
  </Tab>
</Tabs>

### Ver idioma atual

```bash
nivo lang get
```

Saída exemplo:

```
Idioma atual: pt (Português)
```

---

## Onde é salvo

O idioma fica salvo em `~/.config/nivo/config.json` junto com a API key. A configuração persiste entre sessões e projetos.

<Note>
  O idioma é uma preferência por máquina, não por projeto. Todos os projetos no mesmo computador usam o mesmo idioma configurado.
</Note>
