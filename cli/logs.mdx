---
title: "nivo logs"
description: "Visualize logs de runtime ou de um deployment específico do seu app."
---

## Uso

```bash
nivo logs                          # logs de runtime do app linkado
nivo logs --app APP_ID             # logs de um app específico
nivo logs --deployment DEP_ID      # logs de um deployment específico
```

## Opções

| Flag | Descrição |
|------|-----------|
| `--app <appId>` | Exibe logs de um app específico. Padrão: app do `.nivo`. |
| `--deployment <id>` | Exibe logs de um deployment ao invés dos logs de runtime. |
