---
title: "Logs"
description: "Veja logs de runtime ou logs de build de um deployment."
---

## Runtime logs

```bash
nivo logs
nivo logs --tail 300
nivo logs --app APP_ID --tail 1000
```

## Deployment logs

```bash
nivo logs --deployment DEPLOYMENT_ID
nivo logs --app APP_ID --deployment DEPLOYMENT_ID
```

## Opcoes

| Flag | Descricao |
|------|-----------|
| `--app <appId>` | App especifico. Padrao: app do `.nivo`. |
| `--deployment <id>` | Mostra logs de build do deployment. |
| `--tail <lines>` | Numero de linhas de runtime logs. Padrao: `300`. |
