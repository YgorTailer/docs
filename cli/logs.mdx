---
title: "nivo logs"
description: "Show runtime logs or deployment logs for your app."
---

## Usage

```bash
nivo logs
nivo logs --deployment DEPLOYMENT_ID
nivo logs --app APP_ID
```

## Options

| Flag | Description |
|------|-------------|
| `--app <appId>` | Show logs for a specific app. Defaults to the app linked in `.nivo`. |
| `--deployment <id>` | Show logs for a specific deployment instead of runtime logs. |
