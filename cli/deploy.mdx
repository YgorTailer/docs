---
title: "Deploy"
description: "Compacta o diretório, envia o ZIP e cria um novo deployment."
---

## Uso

```bash
nivo deploy              # deploya o diretório atual
nivo deploy .            # mesmo que acima
nivo deploy ./dist       # deploya um diretório específico
nivo deploy --watch      # acompanha os logs até o deploy terminar
nivo deploy --app ID     # deploya para um app específico
```

## Opções

| Flag | Descrição |
|------|-----------|
| `--app <appId>` | Deploy para um app específico (sem ler o `.nivo`). |
| `--watch` | Exibe os logs do deployment no terminal até concluir. |

## Campos do `.nivo` usados no deploy

```json .nivo
{

  "startCmd": "npm start"

```

Veja a [referência completa do `.nivo`](/config) para todos os campos disponíveis.

## Fluxo recomendado

```bash
# Primeira vez: linkar o projeto
nivo link

# Todos os deploys seguintes
nivo deploy . --watch
```
