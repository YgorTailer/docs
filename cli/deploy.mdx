---
title: "Deploy"
description: "Envie codigo ZIP ou dispare redeploy GitHub com checklist visual no terminal."
---

## Uso

```bash
nivo deploy
nivo deploy .
nivo deploy ./dist
nivo deploy --watch
nivo deploy --app APP_ID --watch
```

## Como funciona

O deploy roda como checklist visual com `listr2`:

1. Resolve o `appId` pelo `--app` ou pelo `.nivo`
2. Aplica as configuracoes locais do `.nivo`
3. Busca o app
4. Se `sourceType` for `zip`, compacta e envia o ZIP
5. Se `sourceType` for `github`, dispara redeploy do repositorio
6. Com `--watch`, acompanha os logs ate terminar

## Opcoes

| Flag | Descricao |
|------|-----------|
| `--app <appId>` | Faz deploy para um app especifico, sem depender do `.nivo`. |
| `--watch` | Mostra logs de build ate o deployment terminar. |

## Arquivos ignorados no ZIP

O CLI nao envia:

- `.git`
- `.nivo`
- `.next`
- `node_modules`
- `.env*`
- arquivos `.log`

## Apps GitHub

Para apps GitHub, `nivo deploy` nao envia ZIP. Ele chama o redeploy do app configurado:

```bash
nivo deploy --watch
```
