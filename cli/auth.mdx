## Login

```bash
nivo login
nivo login --manual
```

`nivo login` inicia um device flow, abre o navegador e mostra um codigo no terminal. Depois da autorizacao, a API key fica salva no storage local do usuario.

`nivo login --manual` pede a API key em um prompt mascarado. Use isso em maquinas onde o browser nao esta disponivel.

<Warning>
  A API key nunca e gravada em `.nivo` e nunca e exibida inteira no terminal.
</Warning>

## Onde a chave fica salva

| Sistema | Caminho |
|---------|---------|
| Windows | `%APPDATA%\nivo\config.json` |
| macOS/Linux | `~/.config/nivo/config.json` |

Voce pode trocar a API usada no login:

```bash
nivo login --api-url https://api.nivo.lat/api
```

## Ver conta

```bash
nivo me
```

A saida mostra email, ID, tipo de usuario e a API key mascarada.

## Logout

```bash
nivo logout
nivo logout --yes
```

`logout` remove a chave local. Para revogar a chave no servidor, use **Dashboard > Configuracoes > CLI**.
