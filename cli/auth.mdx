---
title: "Autenticação"
description: "Como fazer login, ver a conta logada e encerrar a sessão no CLI do Nivo."
---

## Login

```bash
nivo login
```

Abre uma janela do navegador na página de autenticação do Nivo. Confirme o código exibido no terminal para concluir.

O token de sessão é salvo localmente em `~/.nivo-auth` e não aparece no dashboard.

## Ver conta logada

```bash
nivo me
```

## Logout

Remove o token de sessão local.

```bash
nivo logout
```

<Note>
  Não commite o arquivo `~/.nivo-auth` no seu repositório — ele contém seu token de acesso.
</Note>
