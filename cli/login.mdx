## nivo login

```bash
nivo login
```

O CLI inicia o device flow:

<Steps>
  <Step title="Código exibido no terminal">
    Um código de 8 caracteres aparece no terminal. Ex: `NIVO-A1B2`.
  </Step>
  <Step title="Navegador abre automaticamente">
    A página de confirmação do Nivo abre no navegador padrão.
  </Step>
  <Step title="Confirme o código">
    Cole ou confirme o código exibido no terminal na página do navegador.
  </Step>
  <Step title="API key salva localmente">
    Após confirmação, a API key é salva em `~/.config/nivo/config.json`.
  </Step>
</Steps>

<Warning>
  Se o navegador não abrir automaticamente, copie e acesse manualmente a URL exibida no terminal.
</Warning>

## nivo me

Verifica a conta autenticada atualmente:

```bash
nivo me
```

Saída exemplo:

```
✔ Autenticado como: joao@exemplo.com
  Chave: nivo_**********************8f3a
  Workspace: minha-empresa
```

---

## nivo logout

Remove a API key do armazenamento local:

```bash
nivo logout
```

<Note>
  O logout não revoga a API key nos servidores do Nivo — apenas remove do armazenamento local. Para revogar a chave, acesse as configurações do dashboard.
</Note>

---

## Onde a chave é salva

| Sistema | Caminho |
|---------|---------|
| Linux / macOS | `~/.config/nivo/config.json` |
| Windows | `%APPDATA%\nivo\config.json` |

<Warning>
  Nunca commite nem compartilhe o arquivo de config — ele contém sua API key pessoal.
</Warning>
