## Installation

```bash
npm install -g @nivocloud/cli
```

## Update

```bash
npm update -g @nivocloud/cli
```

## Get started

<Steps>
  <Step title="Authenticate">
    ```bash
    nivo login
    ```
    Opens a browser window to authenticate. Confirm the code shown in the terminal.
  </Step>
  <Step title="Link your project">
    ```bash
    nivo link
    ```
    Reads your `.nivo` config, creates (or updates) the app, and saves the `appId`.
  </Step>
  <Step title="Deploy">
    ```bash
    nivo deploy . --watch
    ```
    Compresses the directory, uploads a ZIP, starts the build, and tails logs until completion.
  </Step>
</Steps>
