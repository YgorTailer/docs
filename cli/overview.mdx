## Instalação

<Tabs>
  <Tab title="npm">
    ```bash
    npm install -g @nivo-lat/cli
    ```
  </Tab>
  <Tab title="pnpm">
    ```bash
    pnpm add -g @nivo-lat/cli
    ```
  </Tab>
  <Tab title="yarn">
    ```bash
    yarn global add @nivo-lat/cli
    ```
  </Tab>
  <Tab title="bun">
    ```bash
    bun add -g @nivo-lat/cli
    ```
  </Tab>
</Tabs>

Verifique a instalação:

```bash
nivo --version
```

## Fluxo básico

<Steps>
  <Step title="Autentique">
    ```bash
    nivo login
    ```
    Abre o navegador, mostra um código de verificação e salva a API key em `~/.config/nivo/config.json`.
  </Step>
  <Step title="Vincule o projeto">
    ```bash
    nivo link
    ```
    O wizard interativo cria o app na plataforma e grava o `appId` no `.nivo`.
  </Step>
  <Step title="Faça deploy">
    ```bash
    nivo deploy . --watch
    ```
    Compacta, faz upload e exibe logs em tempo real até o deploy terminar.
  </Step>
</Steps>

## Todos os comandos

<AccordionGroup>
  <Accordion title="🔐 Autenticação">
    | Comando | Descrição |
    |---------|-----------|
    | `nivo login` | Login por device flow — abre o navegador. |
    | `nivo logout` | Remove a API key local. |
    | `nivo me` | Mostra conta logada e chave mascarada. |
  </Accordion>
  <Accordion title="🚀 Projeto & Deploy">
    | Comando | Descrição |
    |---------|-----------|
    | `nivo link` | Cria ou atualiza o app e salva `.nivo`. |
    | `nivo deploy [dir]` | Deploy do diretório (padrão: `.`). |
    | `nivo deploy . --watch` | Deploy com streaming de logs ao vivo. |
  </Accordion>
  <Accordion title="📦 Apps">
    | Comando | Descrição |
    |---------|-----------|
    | `nivo apps` | Lista todos os apps do projeto em tabela. |
  </Accordion>
  <Accordion title="📋 Deployments">
    | Comando | Descrição |
    |---------|-----------|
    | `nivo deployments` | Lista os últimos deployments do app vinculado. |
  </Accordion>
  <Accordion title="📄 Logs">
    | Comando | Descrição |
    |---------|-----------|
    | `nivo logs` | Últimos 100 logs do container. |
    | `nivo logs --tail 300` | Aumenta o número de linhas. |
    | `nivo logs --follow` | Streaming contínuo de logs. |
  </Accordion>
  <Accordion title="⚙️ Configurações">
    | Comando | Descrição |
    |---------|-----------|
    | `nivo lang set pt` | Define português como idioma. |
    | `nivo lang set en` | Define inglês como idioma. |
    | `nivo lang get` | Exibe o idioma atual. |
  </Accordion>
</AccordionGroup>

## Onde a config é salva

| Arquivo | Conteúdo | Commitar? |
|---------|----------|-----------|
| `~/.config/nivo/config.json` | API key da conta | ❌ Nunca |
| `.nivo` (raiz do projeto) | Config do app | ✅ Pode |

<Warning>
  Nunca adicione `~/.config/nivo/config.json` ao git — ele contém sua API key pessoal.
</Warning>
