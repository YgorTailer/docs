## Instalacao

```bash
npm install -g @nivo-lat/cli
```

O pacote instala o binario `nivo`.

## Idioma

A ajuda e os prompts do CLI tem suporte a portugues e ingles.

```bash
nivo lang set pt
nivo lang set en
nivo lang get
```

O idioma padrao fica salvo neste computador.

## Fluxo basico

<Steps>
  <Step title="Autentique">
    ```bash
    nivo login
    ```
    O CLI abre o navegador, mostra um codigo e salva a API key no storage local do usuario.
  </Step>
  <Step title="Vincule o projeto">
    ```bash
    nivo link
    ```
    O wizard cria ou atualiza o app e grava apenas metadados no `.nivo`.
  </Step>
  <Step title="Faca deploy">
    ```bash
    nivo deploy --watch
    ```
    Apps ZIP enviam a pasta atual. Apps GitHub disparam redeploy do repositorio configurado.
  </Step>
</Steps>

## Comandos

| Comando | Uso |
|---------|-----|
| `nivo login` | Login por device flow. |
| `nivo logout` | Remove a API key local. |
| `nivo me` | Mostra a conta logada e a chave mascarada. |
| `nivo lang set pt` | Salva portugues como idioma padrao. |
| `nivo lang set en` | Salva ingles como idioma padrao. |
| `nivo link` | Cria/atualiza o app e salva `.nivo`. |
| `nivo deploy --watch` | Cria deployment e acompanha logs. |
| `nivo apps` | Lista apps em tabela. |
| `nivo deployments` | Lista deploys do app. |
| `nivo logs --tail 300` | Mostra logs de runtime. |

## UX do terminal

O CLI usa `picocolors` para destaque, `ora` para spinners curtos, `listr2` para checklist do deploy, `@inquirer/prompts` para inputs, `cli-table3` para tabelas, `boxen` para resumos e `terminal-link` para URLs clicaveis quando o terminal suporta.
