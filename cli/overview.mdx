## Instalação

```bash
npm install -g @nivocloud/cli
```

## Atualizar

```bash
npm update -g @nivocloud/cli
```

## Fluxo básico

<Steps>
  <Step title="Autentique-se">
    ```bash
    nivo login
    ```
    Abre o navegador para autenticação. Confirme o código exibido no terminal.
  </Step>
  <Step title="Configure o `.nivo`">
    Crie um arquivo `.nivo` na raiz do seu projeto com `name`, `projectId` e as configurações de build.
  </Step>
  <Step title="Linke o projeto">
    ```bash
    nivo link
    ```
    Cria o app no Nivo e salva o `appId` no arquivo `.nivo`.
  </Step>
  <Step title="Faça o deploy">
    ```bash
    nivo deploy . --watch
    ```
    Envia o código, inicia o build e exibe os logs até concluir.
  </Step>
</Steps>

## Comandos disponíveis

| Comando | Descrição |
|---------|-----------|
| `nivo login` | Autenticar com o Nivo |
| `nivo logout` | Encerrar sessão |
| `nivo me` | Ver conta logada |
| `nivo link` | Linkar projeto local ao Nivo |
| `nivo deploy` | Fazer deploy do app |
| `nivo apps` | Listar apps acessíveis |
| `nivo deployments` | Listar deployments do app |
| `nivo logs` | Ver logs do app |
