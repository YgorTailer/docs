---
title: "nivo deployments"
description: "List deployments for the linked app."
---

## Usage

```bash
nivo deployments
nivo deployments --app APP_ID
```

## Options

| Flag | Description |
|------|-------------|
| `--app <appId>` | List deployments for a specific app. Defaults to the app linked in `.nivo`. |

Lists the most recent deployments with their status, timestamps, and deployment IDs.
