---
title: "Linkar Projeto"
description: "Lê o .nivo, cria ou atualiza o app no Nivo e salva o appId de volta no arquivo."
---

## Uso

```bash
nivo link
```

Lê o arquivo `.nivo` no diretório atual, cria (ou atualiza) o app na plataforma e escreve o `appId` gerado de volta no `.nivo`.

## `.nivo` mínimo necessário

```json .nivo
{
  "name": "meu-api",
  "projectId": "SEU_PROJECT_ID",
  "runtime": "node20",
  "startCmd": "npm start"
}
```

## Resultado após o link

```json .nivo
{
  "name": "meu-api",
  "projectId": "SEU_PROJECT_ID",
  "runtime": "node20",
  "startCmd": "npm start",
  "appId": "77faa1f9-316f-4dbb-8c1c-41c8eb871346",
  "linkedAt": "2026-01-15T12:00:00.000Z"
}
```

Veja a [referência completa do `.nivo`](/config) para todos os campos disponíveis.
