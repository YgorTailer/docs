---
title: "nivo apps"
description: "Lista os apps acessíveis e exibe seus IDs para uso em deploys."
---

## Uso

```bash
nivo apps
```

## Saída de exemplo

```
nome          status    fonte   url
testapp       running   zip     https://testapp.app.nivo.lat
77faa1f9-316f-4dbb-8c1c-41c8eb871346
```

A segunda linha exibe o ID do app. Use-o no `.nivo` ou com a flag `--app` nos outros comandos.

## Usando o ID no `.nivo`

```json .nivo
{
  "appId": "77faa1f9-316f-4dbb-8c1c-41c8eb871346"
}
```
