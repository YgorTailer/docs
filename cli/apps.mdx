---
title: "Apps"
description: "Liste os apps acessiveis pela conta autenticada."
---

## Uso

```bash
nivo apps
```

## Saida

A saida usa tabela Unicode com colunas para nome, status, fonte, runtime, URL e ID.

```text
Name       Status    Source   Runtime   URL                         ID
sales-bot  running   zip      node20    https://sales-bot.nivo.lat   app_123
```

Use o ID com:

```bash
nivo deploy --app app_123 --watch
nivo deployments --app app_123
nivo logs --app app_123
```
