## Installation

<Tabs>
  <Tab title="npm">
    ```bash
    npm install -g @nivo-lat/cli
    ```
  </Tab>
  <Tab title="pnpm">
    ```bash
    pnpm add -g @nivo-lat/cli
    ```
  </Tab>
  <Tab title="yarn">
    ```bash
    yarn global add @nivo-lat/cli
    ```
  </Tab>
  <Tab title="bun">
    ```bash
    bun add -g @nivo-lat/cli
    ```
  </Tab>
</Tabs>

Verify the installation:

```bash
nivo --version
```

## Basic flow

<Steps>
  <Step title="Authenticate">
    ```bash
    nivo login
    ```
    Opens the browser, shows a verification code and saves the API key to `~/.config/nivo/config.json`.
  </Step>
  <Step title="Link the project">
    ```bash
    nivo link
    ```
    The interactive wizard creates the app on the platform and writes the `appId` to `.nivo`.
  </Step>
  <Step title="Deploy">
    ```bash
    nivo deploy . --watch
    ```
    Compresses, uploads and streams logs in real time until the deploy finishes.
  </Step>
</Steps>

## All commands

<AccordionGroup>
  <Accordion title="🔐 Authentication">
    | Command | Description |
    |---------|-------------|
    | `nivo login` | Login via device flow — opens the browser. |
    | `nivo logout` | Removes the local API key. |
    | `nivo me` | Shows the logged-in account and masked key. |
  </Accordion>
  <Accordion title="🚀 Project & Deploy">
    | Command | Description |
    |---------|-------------|
    | `nivo link` | Creates or updates the app and saves `.nivo`. |
    | `nivo deploy [dir]` | Deploys the directory (default: `.`). |
    | `nivo deploy . --watch` | Deploy with live log streaming. |
  </Accordion>
  <Accordion title="📦 Apps">
    | Command | Description |
    |---------|-------------|
    | `nivo apps` | Lists all project apps in a table. |
  </Accordion>
  <Accordion title="📋 Deployments">
    | Command | Description |
    |---------|-------------|
    | `nivo deployments` | Lists the latest deployments of the linked app. |
  </Accordion>
  <Accordion title="📄 Logs">
    | Command | Description |
    |---------|-------------|
    | `nivo logs` | Last 100 container logs. |
    | `nivo logs --tail 300` | Increases the number of lines. |
    | `nivo logs --follow` | Continuous log streaming. |
  </Accordion>
  <Accordion title="⚙️ Settings">
    | Command | Description |
    |---------|-------------|
    | `nivo lang set pt` | Sets Portuguese as the language. |
    | `nivo lang set en` | Sets English as the language. |
    | `nivo lang get` | Shows the current language. |
  </Accordion>
</AccordionGroup>

## Where config is stored

| File | Contents | Commit? |
|------|----------|---------|
| `~/.config/nivo/config.json` | Account API key | ❌ Never |
| `.nivo` (project root) | App config | ✅ Yes |

<Warning>
  Never add `~/.config/nivo/config.json` to git — it contains your personal API key.
</Warning>
