## nivo deployments

```bash
nivo deployments
```

Lists the latest deployments of the app configured in `.nivo` in a formatted table.

### Deployment statuses

| Status | Description |
|--------|-------------|
| `queued` | Waiting in the build queue. |
| `building` | Build in progress. |
| `waiting_capacity` | Waiting for an available node. |
| `success` | Deploy completed successfully. |
| `error` | Build or startup failure. |
| `cancelled` | Deploy cancelled. |

<Tip>
  To see the logs of a specific deployment, use the dashboard or follow along during the deploy with `nivo deploy . --watch`.
</Tip>
