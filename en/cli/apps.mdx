## nivo apps

```bash
nivo apps
```

Lists all apps of the linked project in a formatted table in the terminal.

### Possible statuses

| Status | Description |
|--------|-------------|
| `running` | App running and responding. |
| `stopped` | App stopped manually. |
| `building` | Build in progress. |
| `error` | Deploy failure or container crash. |
| `starting` | Container starting up. |

<Note>
  The command uses the `projectId` from the `.nivo` file in the current directory. Make sure you're in the project directory.
</Note>
